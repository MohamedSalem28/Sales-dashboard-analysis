# 📊 Sales Performance Dashboard

## 📌 Overview
This project presents an interactive Sales Dashboard built using Power BI to analyze revenue, cost, and profitability across different product categories.

## 🛠️ Tools Used
- Power BI

## 📈 Key Metrics
- Total Sales
- Total Cost
- Total Profit
- Profitability
- Total Quantity
- Chillers Quantity
- Dry Quantity

## 🔍 Key Insights
- Profitability analysis helps evaluate business performance beyond just revenue.
- Chillers and Dry categories contribute differently to total sales and profit.
- Cost vs Sales comparison highlights efficiency and profit margins.
- Quantity distribution reveals demand patterns across product types.
